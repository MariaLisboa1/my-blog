## Getting Started

clone this repository: `git clone https://github.com/MariaLisboa1/my-blog.git`

### Prerequisites

1 - Node.js 10.9.0 or later

2 - Npm package manager

### Installing

1 - `npm install -g gatsby-cli`

2 - `npm install`

### Run the Server

1 - `gatsby develop`
